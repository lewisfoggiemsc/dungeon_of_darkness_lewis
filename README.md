# Dungeon_of_Darkness
An RPG game created by Ted Irland and modified by Lewis Foggie.

## Installation & Usage
Run from terminal on macOS:

### Clone and install dependencies
```bash
git clone https://github.com/jamesf1992/dungeon_of_darkness_lewis
cd dungeon_of_darkness_lewis
uv sync
source .venv/bin/activate
```

### New Branch
git checkout -b new_feature_

### add
git add main.py
git commit -m 
git push origin new_feature_

### Run game
```bash
python3 main.py
```
