# Dungeon_of_Darkness
An RPG game created by Ted Irland and modified by Lewis Foggie.

## Installation & Usage
Run from terminal on macOS:

### Clone and install dependencies
```bash
git clone https://github.com/jamesf1992/dungeon_of_darkness_lewis
cd dungeon_of_darkness_lewis
uv sync
source .venv/bin/activate
```

### New Branch
```bash
git checkout -b new_feature_
```

### add
```bash
git add main.py
git commit -m 
git push origin new_feature_
```

### Run game
```bash
python3 main.py
```

## Run semgrep
```bash
semgrep ci
```

## Prompt
```bash
Change the title of the "Start" button to "Start Dungeon" and change the size of the button according to the extra text proportions. Do not install any new modules or run any commands.
```
